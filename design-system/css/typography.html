Formatting

Style Size Content
Bold 14px Use the <STRONG> to emphasis text.
Italics 14px Use the <EM> to italicize text.
Small 12px Small text <SMALL> is used primarily as help text under form fields and as secondary supporting text in applications and should be used sparingly.
Link 14px Use when you are linking just a few words of text, or when a link is standalone. For lists of objects that are not mixed with paragraphs, such as a list of recent issues or commits in a side panel, use the default neutral color.Unless otherwise specified, hyperlinks use color B400 and are not underlined except on hover.For user generated content, changing the color choice for a whole element, for example an H2 heading, will still display in the default hyperlink color. If the user can apply a color choice directly to a link, then the product should respect that new color choice.
